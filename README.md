# Admin_quanlyve_maybay
Trang quản trị (Admin) của hệ thống bán vé máy bay hỗ trợ quản lý dữ liệu chuyến bay, vé và khách hàng thông qua các thao tác CRUD. Hệ thống tích hợp tính giá vé tự động, gửi email thông báo và cung cấp thống kê báo cáo, đảm bảo dữ liệu chính xác và vận hành ổn định.
Chức năng chính
Quản lý dữ liệu chuyến bay, vé và khách hàng (CRUD)
Tính giá vé tự động dựa trên các điều kiện hệ thống
Gửi email thông báo đến khách hàng
Thống kê và báo cáo tình hình bán vé
Đảm bảo tính chính xác và nhất quán của dữ liệu
Vai trò
Thành viên dự án (Project Member)
Phát triển và quản lý module Admin hệ thống

PHP
SQL Server / MySQL
HTML, CSS, JavaScript
